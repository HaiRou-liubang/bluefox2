\subsection{详细描述}
A class to configure the behaviour of trigger signals. 

Features in this class will open up a more flexible way to configure certain trigger events for devices then the previous features('trigger\+Interface', 'trigger\+Source' and 'trigger\+Mode' used.

\begin{DoxyNote}{注解}
Not ever device will support every property listed in this class for every trigger mode. Always call {\bfseries \hyperlink{classmv_i_m_p_a_c_t_1_1acquire_1_1_component_ac51e55e7e046101f3c6119d84123abd5}{mv\+I\+M\+P\+A\+C\+T\+::acquire\+::\+Component\+::is\+Valid}} to check whether a feature is available or not before using it. Otherwise an exception will be generated. 
\end{DoxyNote}
