\subsection{详细描述}
A wrapper class to handle {\bfseries mv\+I\+M\+P\+A\+C\+T\+::acquire\+::\+Image\+Buffer} structures. 

This class acts as a simple wrapper to handle {\bfseries mv\+I\+M\+P\+A\+C\+T\+::acquire\+::\+Image\+Buffer} structures. When using this class the user is no longer responsible for the memory management of {\bfseries mv\+I\+M\+P\+A\+C\+T\+::acquire\+::\+Image\+Buffer} structures.

Instances of this class are reference counted. Memory is freed once the last instance to it is destroyed. For details read the corresponding sections in the documentation of the various constructors and function belonging to this class.
