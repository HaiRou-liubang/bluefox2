\subsection{详细描述}
A base class for exceptions related to the property module. 

This exception object will only be raised if an unknown error occurred. To check for errors related to the property module a catch block for {\bfseries \hyperlink{classmv_i_m_p_a_c_t_1_1acquire_1_1_e_property_handling}{mv\+I\+M\+P\+A\+C\+T\+::acquire\+::\+E\+Property\+Handling}} objects can be written to catch all property module related exceptions.
