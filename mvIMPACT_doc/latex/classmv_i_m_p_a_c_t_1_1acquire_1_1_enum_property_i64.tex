A template class to represent 64 bit integer properties and enumerated 64 bit integer properties. 

The template parameter specifies the enum to use as the value type for this enumerated property. The class itself however is restricted for integer values. To use a normal int (not enumerated) properties use the {\bfseries \hyperlink{group___common_interface_ga81749b2696755513663492664a18a893}{mv\+I\+M\+P\+A\+C\+T\+::acquire\+::\+Property\+I64}} type.
