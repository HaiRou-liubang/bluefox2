详细描述

A base class for mvIMPACT::acquire::Method object related exceptions from the property module.

This exception object will never be raised in the code. It can't be constructed directly but to check for mvIMPACT::acquire::Method object related errors a catch block for mvIMPACT::acquire::EMethod objects can be written to catch only Method related exceptions.
