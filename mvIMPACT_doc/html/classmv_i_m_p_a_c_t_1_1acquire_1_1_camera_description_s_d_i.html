详细描述

A base class to describe a SDI(Serial Digital Interface) camera signal (Device specific interface layout only).

This class provides access to all settings specific for SDI video signals.

注解
This class will only be available if mvIMPACT::acquire::Device::interfaceLayout is set to mvIMPACT::acquire::dilDeviceSpecific before the device is opened.

类成员变量说明

PropertyIVideoStandard mvIMPACT::acquire::CameraDescriptionSDI::videoStandard

An enumerated integer property for defining the video standard this camera is compliant with.

Valid values for this property are defined by the enumeration mvIMPACT::acquire::TVideoStandard.
