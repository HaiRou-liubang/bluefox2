详细描述

A base class that provides access to the most common settings for a device(Device specific interface layout only).

Use one of the class derived from this class to get access to all the available settings.

注解
This class will only be available if mvIMPACT::acquire::Device::interfaceLayout is set to mvIMPACT::acquire::dilDeviceSpecific before the device is opened.
