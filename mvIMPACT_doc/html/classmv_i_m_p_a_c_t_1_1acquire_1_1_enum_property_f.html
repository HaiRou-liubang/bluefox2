template<class ZYX>
class mvIMPACT::acquire::EnumPropertyF< ZYX >

A template class to represent float properties and enumerated float properties.

The template parameter specifies the enum to use as the value type for this enumerated property. The class itself however is restricted for float values. To use a normal float (not enumerated) properties use the mvIMPACT::acquire::PropertyF type.

const EnumPropertyF& mvIMPACT::acquire::EnumPropertyF< ZYX >::getTranslationDict ( std::vector< std::pair< std::string, ZYX > > &  sequence) const
inline

This function queries the property's translation table.

If this property defines a translation table the strings and their corresponding translation values will be written into sequence. If the property does NOT define a translation table sequence will be empty after this function call.

注解
This function is much more efficient than calling EnumPropertyF::getTranslationDictString and EnumPropertyF::getTranslationDictValue multiple times and therefore this function should be called whenever all entries are required.
返回
A const reference to the calling property.
参数
[out]sequenceA reference to a container which will receive the data from the properties translation dictionary.
